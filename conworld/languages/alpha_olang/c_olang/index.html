アルファ祖語(仮)からの音韻変化

規則

  • b",d",g" → v,d'[ð],x' / _
  • {td,dt},{kg,gk},{pb,bp} → tt,kk,pp / _
  • p t k → f ts x/ _#
  • x,x' → a / _(only stressed syllable)
  • h,x,x' → ∅ / _(only NOT stressed syllable)
  • h → ∅ / _{$,C}
  • e[ɛ],o[ɔ] → e[e],o[o] / _
  • u',o'[œ],q' → i,e'[ɛ],o'[ɔ] / _
  • a[a] → a'[æ] / _
  • q → a[a] / _
  • w → v / _
  • i → hi / #_
  • t → ts _i
  • tt → ts /_
  • s,c → z,j / V_{V,#}
  • c → s / _
  • y → j / {#,V}_
  • y → ∅ / _
  • a → ∅ / _a'
  • e → ∅ / _e'
  • o → ∅ / _o'

正書法

ラテン文字

　正式な文字(制作途中)をラテン文字で転写している。
  • tiは常にtsiなのでtiと表記
  • e'・o'・a'はマクロンをつけて表記 ē ō ā
  • d'はðで表記
  • 長子音は子音を2つ重ねる
  • 形態素の維持のためにaā・eē・oōと表記することがあるが、読みはa'・e'・o'

文法

名詞

数

　双数形は複数形に吸収された。

格

　アルファ祖語(仮)から、具格と処格が奪格に統合した。 属格は祖語と同様にない。
