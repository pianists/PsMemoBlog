フレロッド・ルセイス祖語

仮想世界アルビスでいう印欧祖語にあたる言語です。
辞書はこちら。

造語法

　一般に言語の解説といえば例文か文法から始まるが、 制作途中でまだ制作が始まったばかりという特殊な状態にあるため造語法を明文化しておく。 が、ある程度自分の主観で作る部分もあるため、 辞書において造語法が「主観」となっている語については例外として扱う。

語根の作成

　基本的に5つの音象徴からランダムに選んで造語する。 5つの音象徴は以下。
soft m n w y p f r j
hard s c f* d* x g" z v
light p k t c x r` d"* t*
heavy b d g z x' r b" d"
neutral l r s x h k b t
　例えばserであれば、hard + 母音 + heavyというように造語すると決め、 serに決まった。 これはサイコロや乱数の値が違かったらcab"であったかもしれないということだ。 1つの子音につき、1つの音象徴があてられる。
　*部分はもともと別の音素だったが、修正したものである。

動詞の作成

　動詞は比較的語根と近い形になる。 語根+子音が一般的であるため、CVCCという音節構造が多い。 追加する子音についても1つの音象徴からランダムで選ぶ。 活用順は不定 -> 進行 -> 完了->完結->将前。

名詞の作成

　純粋な名詞はあまり作らない方針。 具体的には、風という名詞を作りたいならば、 風が吹くという動詞を作ってから動詞派生語の接尾辞og"で風という名詞にする。 曲用順は単->主->対->与->奪->具->処。

文法

形態論的分類

　SOMV・AN・後置詞の屈折語。 主題優勢言語で、統語的に主語は必要ない。 主題と主格を区別するために、以下主題のことを題目と言う。

名詞

格

　名詞には格が8つある。 単独格、主格、題目格、対格、与格、奪格、具格、処格である。 ただし、一般名詞・固有名詞では題目格の曲用がなく、題目構文を使う。 代名詞には題目格の曲用がある。 無標は単独格である。
　主題優勢言語のため、主格はあまり用いられず、 題目構文を使うことが専らである。 (他の主題優勢言語と同じく、名詞節などのメインでない文では主格が用いられる。)
　具格は道具と人両方に使え、英:withとほぼ同じ用法だ。
　処格は場所、時間両方に使え、位置を表す名詞ともに用いられることが多い。 　曲用の不規則さは名詞によって違うが、人称代名詞は最も不規則で、 その他は規則的なものが多い。 固有名詞はほとんど規則的で、 主・対・与・奪・具・処の順に、im・em・ems・t・emd"・emicの接尾辞がつく。

数

　名詞は単数・双数・複数の区別が必須である。ただし、定不定の区別はない。 基本的に接尾辞のotで双数、epで複数を表すが、 二音節以上の単語では、二音節目が母音交替することもある。

動詞

相(アスペクト)

　時制がないため動詞は相によってのみ活用する。 不定相、進行相、完了相、完結相、将前相があり、母音交替と接尾辞により活用する。 不定相は、英語などの現在形にあたり、習慣や一般事象について述べる(習慣相)的な使い方と 相に重点を置かないコピュラ動詞系の語に使う。 進行相は進行形(〜している)にあたり、 完了相は動作を流れで見たときに完了した時、 完結相は点事象で、完了した出来事を一瞬で終わったように扱う。 将前相は完了相の逆で、まだしていないことを表す。 　活用は不定を基礎として進行、完了の順に母音交替し、
