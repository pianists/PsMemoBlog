規則

  • b",d",g" → v,d'[ð],x' / _
  • x,x' → a / _(only stressed syllable)
  • h,x,x' → ∅ / _(only NOT stressed syllable)
  • h → ∅ / _$
  • e[ɛ],o[ɔ] → e[e],o[o] / _
  • u',o'[œ],q' → i,e'[ɛ],o'[ɔ] / _
  • a[a] → a'[æ] / _
  • w → v / _
  • i → hi / #_
  • sy,ty,zy,dy → s,t,z,d / _
  • t → ts _i
  • s,c → z,j / V_{V,#}
  • c → s / _
  • y → j / {#,V}_
  • y → ∅ / _
